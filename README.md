## ✨ Key Features
* **Modern UI:** Crafted with Tailwind CSS & AOS (Animate on Scroll).
* **Automated Logic:** Real-time application handling with unique ID generation.
* **Admin Control:** Secure dashboard to approve/reject applications.
* **Tracking:** Live status tracking via Application ID or Aadhar.

## 🛠️ Tech Stack
* **Backend:** Flask (Python)
* **Database:** MySQL
* **Frontend:** HTML5, Tailwind CSS, JavaScript
